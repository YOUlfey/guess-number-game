## Simple network game "Guess the number"
The first player imagine the number. The Second player try to guess the number.

### Launch game

1. Build jar package
```
mvn clean install
```
2. Run jar package
```
java -jar target/guess-the-number-game-0.0.1-SNAPSHOT.jar
```
3. Go to site
```
localhost:9090
```
